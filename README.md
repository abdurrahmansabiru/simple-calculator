# Simple Calculator in Python

## Overview

This is a simple Python calculator project that performs basic arithmetic operations such as:

- Addition
- Subtraction
- Multiplication
- Division

The program is menu-driven and accepts user input to perform the selected operation.

---

## Features

- Simple and beginner-friendly
- Uses Python functions
- Handles division by zero
- Input-based operation selection
- Displays appropriate error messages for invalid choices

---

## Technologies Used

- Python 3

---

## Functions Used

| Function | Description |
|----------|-------------|
| add(a, b) | Returns addition of two numbers |
| subtract(a, b) | Returns subtraction of two numbers |
| multiply(a, b) | Returns multiplication of two numbers |
| division(a, b) | Returns division of two numbers |

---

## Program Workflow

1. User enters the first number
2. User enters the second number
3. Menu is displayed
4. User selects an operation
5. Result is displayed
